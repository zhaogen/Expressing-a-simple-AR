주요 기능
카메라 자세 추정 (Pose Estimation): cv.solvePnP를 사용하여 실시간 영상 내 체스판의 위치와 각도(R, t)를 계산합니다.

AR 물체 렌더링: cv.projectPoints를 통해 3D 정육면체 좌표를 2D 화면으로 투영하여 시각화합니다.

커스텀 시각화: 정육면체의 바닥, 기둥, 윗면의 색상을 다르게 설정하여 입체감을 강조했습니다.
